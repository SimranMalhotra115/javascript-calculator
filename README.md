# JavaScript Calculator

A simple and interactive **JavaScript Calculator** built using **React**.  
This project performs basic arithmetic operations and follows the functionality required in the freeCodeCamp **JavaScript Calculator** challenge.

## Features

- Number buttons from `0` to `9`
- Basic operators: `+`, `-`, `*`, `/`
- Decimal point support
- Clear button to reset calculator
- Formula display and current value display
- Handles chained operations
- Prevents multiple leading zeros
- Prevents multiple decimals in one number
- Supports negative numbers after operators
- Continues calculation using previous result after pressing `=`

## Technologies Used

- HTML
- CSS
- JavaScript
- React

- Operator buttons with ids:
  - `add`

- `id="clear"`
- `id="display"`

It also supports:

- Clearing all input and output values
- Decimal calculations
- Consecutive operator handling
- Previous result reuse after evaluation
- Reasonable decimal precision

## How to Run the Project

1. Download or clone the project
2. Open the folder in VS Code
3. Save the code in a file named `index.html`
4. Open the file in your browser

You can also use the **Live Server** extension in VS Code for a better experience.
